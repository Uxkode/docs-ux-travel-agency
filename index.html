UX Travel Agency Management System

UX Travel Agency is a comprehensive web-based management system designed for travel agencies to streamline operations, manage customers and agents, track finances, and generate professional invoices and reports ‚Äî all from one intuitive dashboard.

Getting Started

Follow these steps to install and set up UX Travel Agency:

  1. Upload the downloaded ux-travel-agency.zip file to your web server directory using cPanel (e.g., public-html/ux-travel-agency.zip or admin.yourdomain.com/ux-travel-agency.zip).
  2. Now extract the zip file and select all files folder in the ux-travel-agency folder and then move all files to the public-html or admin.yourdomain.com main directory.
  3. Create a MySQL database and import the schema.sql file to create all necessary database tables.
  4. Edit the config.php file and then first update the database name, database user and password information correctly.
  5. Set the correct APP_URL in config.php to match your installation path.
  6. Ensure the uploads/ directory is writable (chmod 755).
  7. Access the application at your configured URL and log in with your admin credentials.

System Requirements

  • PHP 7.4 or later
  • MySQL 5.7 or later
  • Apache web server with mod_rewrite enabled
  • Modern web browser with JavaScript enabled
  • Minimum 10MB disk space for application files

Customer Management

Manage all your travel customers from a centralized location with complete contact information and booking history.

  1. Go to Customers from the main menu.
  2. Click Add New Customer to create a new customer profile.
  3. Fill in customer details including name, email, phone, and address.
  4. Save the customer profile to track their bookings and interactions.
  5. Click on a customer to view their complete profile and associated bookings.
  6. Edit or delete customer information as needed.

Tip: Keep customer information up-to-date for better communication and booking accuracy.

Agent Management

Manage your travel agents and track their performance, commissions, and activities.

  1. Navigate to Agents from the sidebar menu.
  2. Click Add New Agent to create a new agent profile.
  3. Enter agent details including name, email, phone, and commission rate.
  4. Assign agents to bookings and track their performance metrics.
  5. View agent earnings and calculate commissions automatically.
  6. Update agent information and commission rates as needed.

Tip: Use agent tracking to monitor performance and optimize your team productivity.

Financial Tracking

Monitor your travel agency's earnings and expenses with detailed tracking and categorization.

Recording Earnings

  1. Go to Earnings from the menu.
  2. Click Add Earning to record a new revenue entry.
  3. Select an earning category (e.g., booking commission, service fee).
  4. Enter the amount, date, and description.
  5. Optionally assign the earning to an agent.
  6. Save to update your revenue records.

Managing Expenses

  1. Navigate to Expenses from the menu.
  2. Click Add Expense to record a new business expense.
  3. Select an expense category (e.g., office supplies, marketing).
  4. Enter the amount, date, and description.
  5. Save the expense record for accounting purposes.

Tip: Regularly update earnings and expenses to maintain accurate financial records and generate meaningful reports.

Invoices & Reports

Generate professional PDF invoices and comprehensive business reports.

Creating Invoices

  1. Go to Reports from the main menu.
  2. Select invoice template and customer information.
  3. Review invoice details (items, amounts, taxes).
  4. Your custom logo (set in Settings) will appear on the invoice.
  5. Download or print the invoice as PDF.

Generating Reports

  1. Navigate to Reports section.
  2. Select report type (earnings, expenses, activity log).
  3. Choose date range and apply filters if needed.
  4. Review detailed report with charts and summaries.
  5. Export or print for record-keeping.

User Management

Create and manage admin and staff user accounts with appropriate access levels.

  1. Go to Users from the admin menu.
  2. Click Add New User to create a new account.
  3. Enter user details: name, email, username, and password.
  4. Save the user account.
  5. Manage user profiles and permissions from the Users dashboard.
  6. Edit user information or reset passwords as needed.
  7. Delete inactive accounts from the system.

Settings & Configuration

Customize your application with company information and branding.

  1. Click Settings from the main menu.
  2. Update your company name, tagline, and contact information.
  3. Set your business address, city, and country.
  4. Configure your application URL and currency.
  5. Upload Custom Logos: App logo, favicon, invoice logo, and login page logo.
  6. Save all changes to apply them across the system.

Logo Guidelines: JPG/PNG formats, recommended sizes for best appearance, max file sizes enforced.

Activity Logs

Track all system activities and user actions for security and accountability.

  1. Navigate to Activity Log from the menu.
  2. View complete audit trail of all system actions.
  3. Filter logs by user, action type, or date range.
  4. Search for specific activities using keywords.
  5. Export logs for compliance and record-keeping.

FAQ

Is the system mobile-friendly?

Yes, UX Travel Agency features a fully responsive design that works seamlessly on desktop, tablet, and mobile devices.

Can I customize invoice templates?

Yes, you can customize invoice appearance including company logo, header information, and footer details from the Settings page.

How do I backup my data?

Regularly backup your database using MySQL administration tools or automated backup scripts. Also backup the entire application folder.

Can I export reports to PDF?

Yes, all reports can be generated as PDF files for easy sharing, printing, and archiving.

How do I reset a forgotten password?

Navigate to the login page and use the "Forgot Password" option to reset your password via email verification.

Troubleshooting

Tip: Always check the activity logs to troubleshoot issues and understand what actions were performed.
  • If you see a "database connection error", verify your database credentials in config.php.
  • If images don't upload, ensure the uploads/ directory has write permissions (chmod 755).
  • If you can't log in, clear your browser cache and cookies, then try again.
  • For PDF generation issues, verify that the uploads/ directory is writable and has sufficient disk space.
  • If features are missing, ensure the database schema was properly imported with all required tables.
  • Check server error logs if the application produces unexpected errors or blank pages.
